Project Introduction

  • Predicting whether or not an auto insurance claim is fraudulent based on historical data
  • It is important to predict fraud to help save companies money. Billions of dollars are lost due to fraud every year, so proactively predicting fraud will help to minimize this loss.
  • Dataset was designed to give students practice with predictive modeling; however, there are many insurance companies using this process to predict fraud within their own company

The Dataset Introduction

  • Was extracted from an insurance companys claim database
  • Case was presented at the 2018 EDSIG Conference on Information Systems & Computing Education
  • 5001 observations
  • Target variable: Fraudulent Claim
  • 17 kept input variables: 11 categorical, 6 numeric

List of Input Variables

  • State Code
  • Claim Amount
  • Education (highest level)
  • Claim Date
  • Employment Status
  • Gender
  • Income
  • Location (Residential, Suburban, Urban)
  • Marital Status
  • Annual Premium
  • Months Since Last Claim
  • Months Since Policy Inception

Input Variables cont.

  • Claim Cause
  • Claim Report Type
  • Vehicle Class
  • Vehicle Size
  • Outsanding Balance (how much is left to pay off on the car)
  • Dropped variables: Claimant Number, State, Monthly Premium, and Vehicle Model

Handling missing values #2

The second way to handle missing values is to impute numeric missing values with the median and catergorical values with the mode

imputer=function(x){
  if(!is.numeric(x)){
    levels=unique(x)
    x[is.na(x)]=levels[which.max(table(x))]
  }
  return(x)
}

median=preProcess(fraud, method='medianImpute')
fraud_med=predict(median, newdata = fraud)
fraud_med=data.frame(lapply(fraud_med,imputer))
sum(is.na(fraud_med))

Handling missing values #3

The third way to handle missing values is to impute numeric missing values with a constant (0 in this case) and catergorical values with the mode

imputer2=function(x){
  if(is.numeric(x)){
    x[is.na(x)]=0
  }else{
    levels=unique(x)
    x[is.na(x)]=levels[which.max(table(x))]
  }
  return(x)
}

Balancing the train data

undersample=function(x){
  train1=x[x[,'target']=="1",]
  n1=nrow(train1)

  train0=x[x[,'target']=="0",]
  n0=nrow(train0)

  train00=train0[sample(1:n0,n1),]
  return (rbind(train00,train1))
  
}

Choosing the best model based on Balanced Accuracy

compare=c()
prediction=function(model, test){
  pred=predict(model,test)
  cm=confusionMatrix(pred,test[,'target'],positive="1")
  compare=c(compare,cm$byClass['Balanced Accuracy'])
}

accuracies=mapply(prediction, models, test_data)
best=max(accuracies)
index=match(best,accuracies)
best_model_name=names[[index]]
best_model=models[[index]]
